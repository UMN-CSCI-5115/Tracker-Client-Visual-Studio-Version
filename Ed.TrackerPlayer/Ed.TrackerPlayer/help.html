Tracker Mobile Help

Tracker is a free video analysis and modeling tool designed to be used in physics education. Tracker video modeling is a powerful new way to combine videos with computer modeling based on the Open Source Physics (OSP) Java framework. For more information see the official Tracker website.

Tracker Mobile is an Open Source project undertaken by edmentum. Currently it allows much of the viewing functionality of the original Tracker program files to be achieved through a web browser, rather than through a stand-alone application. While Tracker Mobile lacks some of Tracker’s features, there is a lot of room for program expansion. Check out the github (link) and feel free to fork it if you want to get involved!

Tracks

A track is used to model an element that changes over time. Tracker Mobile can currently show two types of tracks.

Point Mass

A point mass represents a zero dimensional point that has some finite mass. They do not exist in reality, as it is not possible for an object to have mass without volume. However, point masses are the building blocks with which more complex and realistic models of physical systems are constructed in classical physics.

Center of Mass

A center of mass represents the center of mass of a collection of point masses. Its mass is defined as the sum of the masses of the points to which it is linked. Similarly, its steps are not manually marked but instead are determined by the positions and masses of its linked point masses. If all of the linked point masses are equal in weight, the center of mass will appear in the exact center of the linked masses. Otherwise the position of the center of mass to be weighted towards the point masses with the most mass.

Track Options

Track options are accessible from the "Tracks" button at the top of the page.

Show or Hide

This option hides or shows the given track on the video. Nothing changes beyond visibility. Since the track still exists, its plot and table information can still be viewed. Showing or hiding a point mass that is linked to a center of mass will not alter the position of the center of mass.

Notes

These represent any information left behind by the instructor. If left blank by the instructor, nothing will be displayed.

Video

Tracker Mobile charts the physical locations of point mass tracks and center of mass tracks as the video progresses. Each track is represented by a distinctive shape and color that allows for easy visualization of the object being modeled. The video consists of a series of individual frames which show the position of each track at a given time.

The number of the current frame being displayed is represented on the left, and the video play speed is on the right.

The number represents the step size, or interval between frames being played. The buttons on the right allow the video to be stepped through on a frame by frame basis (image). The button on the far right toggles video looping.

Windows

Tracker Mobile currently supports viewing track properties through two view types: plots and tables.

Plots

A plot charts an x parameter against a y parameter in the same color as the selected track. The x and y parameters can be changed by clicking the button next to each respective axis.

The data point associated with the video frame currently being displayed is highlighted as the video plays. Clicking on a plot point while the video is paused will cause the video to jump to the frame associated with that data.

Zooming is achieved by clicking and dragging over the desired area. Double clicking on the plot will zoom back out to the default zoom.

Windows can also be resized by dragging on their containing bars, or toggled to open and close by clicking on the bars.

Tables

Tables can display parameter information about any track. Parameters can be changed by clicking the “Table Options” button.

Clicking on any of the column headers will sort the table by that parameter. The table row associated with the video frame currently being displayed is highlighted as the video plays.

Coordinate Axis

The coordinate axis is the reference point for all tracked points. Its positive x-axis is denoted by a small crossbar.

The coordinate axis visibility can be toggled on or off using its menu button.
